1. What types of native features (on Android and iOS devices) would you be able to implement with the use of HTML/CSS/JS only?

I just want to say, PhoneGap is amazing. The way that you can access certain API's is amazing all capsuled in to 3 things, HTML, CSS, and JS. With PhoneGap, you can access the Camera, Contact, Geolocation and other great features in a specific device, without necessarily knowing the programming language of that device. It's a good thing to know, just like we learned to know JavaScript first before learning jQuery Mobile, because if you stuck with simple jQuery mobile, makes you learn very ugly javascript. So learning those best practices in earning javascript is good for the future of everything else you want to develop for.

2. For your CRUD app revision and/or Demo App components, which of these do you think you will implement?

For my CRUD app revision, I would like to incorporate the Alerts feature, and the Geolocation feature. I would also like to incorporate the Camera component and the Contacts component into my CRUD app. It will be amazing once I am done incorporating these components.

3. Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

I think some lessons learned that happened with me is that I learned not to design and develop before researching. That was the first thing we learned coming into this degree program is to research before you start your design. Another lesson I learned is development after design. I would say I would like to get the programming portion done before the design, because the first thing you learned from user experience is bugs and errors are not a good thing for users, and no matter how good the app looks, they will not come back to the app because of its usability.

4. Please describe any design intentions you may have for the AVF Demo app and the revised/new version of your CRUD app.

Some of my design intentions for the new version, is a much better cool flow. I want a cool feeling, such as sky blue colors, light green colors. Arial font and Helvetica. I want the background color to be sutle to the eye. I know that my app right now is very plan and very basic as far as implementation of jQuery Mobile. But rest assured I will have this app looking like a professional. Believe that!
